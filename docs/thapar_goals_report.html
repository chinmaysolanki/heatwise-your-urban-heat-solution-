Details of Project Semester Assignment:
The project involves the end-to-end design, development, and deployment of HeatWise — a full-stack AI-powered platform
that recommends optimized green cover solutions for urban rooftops in Indian cities to combat the Urban Heat Island (UHI)
effect. The platform integrates a hybrid Machine Learning and rules-based recommendation engine, AR-assisted space
measurement, 3D garden layout visualization, a verified green installer marketplace, and longitudinal outcome tracking.
During this project semester, the student is responsible for: (1) Frontend development of all 27 application screens
using Next.js and React, implementing the "Midnight Aurora" dark UI theme with 3D animations via Three.js and
CSS animations; (2) Building and integrating the real-time Notifications and Alerts system for heat alerts, AI analysis
updates, installer replies, eco tips, and achievement milestones; (3) Designing Flutter-based UI components using the
established Midnight Aurora design token system (color palette: #04091A background, #38BDF8 primary electric blue,
#14B8A6 teal accent) and porting them to the web application; (4) Integrating the recommendation engine frontend
with the backend API (/api/recommendations/generate) that orchestrates ML scoring, catalog hybrid fallback, supply
constraint checking, and geospatial enrichment; (5) Supporting the Android Capacitor mobile build for Play Store
submission. The project directly addresses climate resilience in Tier 1–3 Indian cities by enabling measurable urban
temperature reduction through data-driven green infrastructure planning and verified installer deployment.
